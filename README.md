# Web share/bán mod game
- Được viết bởi con gà học web 1 tháng :))))
- Web cùi lắm đừng coi :>

## Giới thiệu
- Sử dụng MySQL và PHP thuần
- Thư viện PHPMailer

## Cài đặt
- Tạo database `web_mod` và import file `web_mod.sql`
- Vô file `conn.php` thay đổi thông tin database
- Vô file `sent-mail.php` thay đổi email gửi, pass, email người nhận và một số thông tin khác
- Cấu hình email để gửi mail. Xem hướng dẫn [tại đây](https://freetuts.net/huong-dan-gui-mail-trong-php-voi-phpmailer-710.html#goto-h2-3).
